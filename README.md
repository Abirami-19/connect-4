# Connect 4 AI

A web-based Connect 4 game where the player competes against an AI opponent. Built with a focus on clean UI, smooth interactions, and strategic gameplay.

---

## Overview

This project recreates the classic Connect 4 game with an AI opponent that can make decisions based on different difficulty levels. It demonstrates grid-based game logic, state management, and basic AI strategies.

---

## Features

* Play against an AI opponent

* Multiple difficulty levels:

  * Easy: random moves
  * Medium: blocks player and basic strategy
  * Hard: smarter decision-making

* 7x6 game board

* Win detection (horizontal, vertical, diagonal)

* Draw detection

* Restart functionality

* Responsive layout

* Smooth UI updates and interactions

## How to Play

1. Click on a column to drop your piece
2. Your piece will fall to the lowest available slot
3. The AI will make its move automatically
4. Connect four pieces in a row to win
